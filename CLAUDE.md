# Simple Calculator (Python CLI)

## Mission
Build a tiny beginner-friendly command-line calculator that:
- Adds
- Subtracts
- Validates input
- Loops until user quits

## Stack
- Python 3.11+
- Standard library only

## Scope
IN:
- + and - only
- CLI menu
- Input validation
- Loop

OUT:
- No GUI
- No web
- No advanced math
- No parsing full expressions

## Rules
- Single file: calculator.py
- Use functions
- Keep it readable
- Handle bad input safely

## How to Run
python calculator.py
